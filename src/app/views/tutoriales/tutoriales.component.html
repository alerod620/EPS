El Circulo se podria definir como el espacio geométrico que se encuentra dentro de una circunferencia.

Dadas 3 longitudes (números, no segmentos), construye un triángulo. Variación de una construcción clásica, excepto que la brújula no se usa para copiar longitudes, sino que se establece en los valores dados.

Se dan tres ejemplos de uso, para adaptarse a diferentes niveles de enseñanza. El video es una mera introducción y hay muchas cosas que puede hacer con un control slide que no se mencionan.

¡Podemos usar las reglas de construcción para copiar un segmento de línea! Esto es muy fácil usando solo las herramientas de brújula y línea.
